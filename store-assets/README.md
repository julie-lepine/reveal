# Assets fiche store (optionnel)

Dossier d’**archivage** pour tes PNG sources — **non servis** par l’app ni par GitHub Pages.

À y mettre (export depuis téléphone ou Figma) :

- Captures **Play Store** (phone 16:9, tablette si besoin)
- Captures **App Store** (6,7", 6,5", iPad… selon exigences Apple)
- Feature graphic Android (1024×500) si tu en fais une

Upload **manuel** dans Play Console et App Store Connect.

Les icône / splash **in-app** sont dans `resources/` (`npm run assets:prepare`).
